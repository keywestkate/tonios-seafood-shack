# Tonio's — Image Assets Needed

Place the following images in this folder. All images should be optimized for web (use WebP where possible).

## Required Images

| File | Dimensions | Description |
|------|-----------|-------------|
| hero-bg.jpg | 2400×1600 min | Main hero — waterfront, dock, exterior or food |
| story-photo.jpg | 1200×900 | Interior bar scene or staff shot |
| gallery-1.jpg through gallery-10.jpg | 800×800 min | Food, drinks, music, people, exterior |
| social-1.jpg through social-5.jpg | 600×600 | Instagram-style square photos |
| og-image.jpg | 1200×630 | Social sharing thumbnail |
| favicon.png | 512×512 | Logo mark for browser tab |
| tonios-logo.png | 600×400 | Original Tonio's logo with transparency |

## Notes
- Hero image should have subject matter on left/center — right side partially obscured by fish illustration
- Gallery grid layout: images 1 and 8 display larger (2-column span) — use strongest shots there
- All images automatically display with fallback brand colors if files are missing
